Add post data or featured image to any part of the website or on each individual post/page.

== Description ==

Add post data on each individual post/page. Very Easy to Implement. Provides you with a shortcode [post-data].


== Installation ==

= Step 1 =

1. Download the zip file
2. Extract the contents
3. Upload the folder `featured-image` to your WP plugin folder `/wp-content/plugins/` directory
4. Go to Plugins > Plugins, and activate the plugin

= Step 2 =

* Use this shortcode '[post-data]' on your page/post content.
* Use this Code on your template inside loop: <?php if ( function_exists('get_post_data') ) get_post_data(); ?>
